# n8n-nodes-olvid-keycloak

[![NPM Version](https://badge.fury.io/js/n8n-nodes-olvid-keycloak.svg?style=flat)](https://npmjs.org/package/n8n-nodes-olvid-keycloak)

This is an n8n community node. It lets you manage your _Olvid Console for Keycloak_ in your n8n workflows.

_Olvid Console for Keycloak_ is a self hosted Olvid directory. It allows registered users to easily get in touch in Olvid, to create and manage Olvid groups and much more.
This node allows you to automate any action in your Olvid management Console with a n8n workflow.

**Olvid** is a private and ethic instant message application available on every platform: https://olvid.io


[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/sustainable-use-license/) workflow automation platform.
